# gosspanel

自用小面板。

## 面板安装

```bash
curl -fsSL https://raw.githubusercontent.com/SadNoo/gosspanel/main/scripts/install-panel.sh | bash
```

## HTTPS 面板安装

```bash
GOSS_HTTPS=1 GOSS_DOMAIN=panel.example.com curl -fsSL https://raw.githubusercontent.com/SadNoo/gosspanel/main/scripts/install-panel.sh | bash
```

GOSS_HTTPS=1 curl -fsSL https://raw.githubusercontent.com/SadNoo/gosspanel/main/scripts/install-panel.sh | bash
